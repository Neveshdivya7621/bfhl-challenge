## ✨ Overview

This project is a **Next.js Monorepo** that combines:

- ⚡ Modern **React Frontend (App Router)**
- 🔌 Scalable **Node.js API**
- 🧠 Advanced **Graph Algorithms Engine**

Built to strictly satisfy all **BFHL evaluation requirements** with **optimized performance and clean architecture**.

## 🧠 Core Features

- ✅ Edge-ready stateless API
- ✅ Multi-parent diamond conflict resolution
- ✅ DFS-based cycle detection
- ✅ Graph clustering & validation
- ✅ Fully responsive UI
- ✅ Glassmorphism + Dark Mode 🌙
- ✅ Animated dashboard (SVG stats)
